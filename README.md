## 🧞 Comandos

| Comando                   | Acción                                           |
| :------------------------ | :----------------------------------------------- |
| `npm install`             | Instalar dependencias                            |
| `npm run dev`             | Inicia servidor de dev en local `localhost:4321` |
| `npm run build`           | Construir el sitio para producción en `./dist/`  |
| `npm run preview`         | Vista previa, antes del deploying                |
